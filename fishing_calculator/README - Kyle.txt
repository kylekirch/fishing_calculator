This is not an executable file, if you want to access the application, it must be hosted through a virtual server.

To avoid many possble issues with execution, follow these steps:

1) Open the folder in VSCode
	- right click the "fishing_calculator - Version 1" folder and select "open with code"

2) Ensure Django/Python is installed
	- In the command prompt, enter "pip3 install django", ensure the download is successfull
	- Do the same for python

3) Run the server
	- In the command prompt, enter "python manage.py runserver", the command prompt should respond with a successfull server IP

4) Enter the IP address in a web browser
